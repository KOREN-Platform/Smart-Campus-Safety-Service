# SmartX Open Platform을 활용한 Smart Campus Safety 서비스 요소기능 개발


Smart Campus Safety 서비스 시나리오에 따른 시연 워크 플로우를 설계하여 분산된 CCTV 단말들로부터 중앙 관제센터로 영상을 전달/저장한 후, 저장된 CCTV 영상에 대한 Deep Learning 기반 영상 프레임 분석 기능을 실증
